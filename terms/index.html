Privacy Policy :#

For this website#

To get stats about the website, I use GoatCounter. It is a privacy-friendly analytics service. It does not track you across the web and respects the Do Not Track header. It does not use cookies and does not store any personal data. It only collects the following information: the number of page views, the number of unique visitors, the country of the visitor, the browser and operating system of the visitor, and the referrer. NO IP ADRESS.

For Pafi#

When you are using the discord bot Pafi, I have basically only acess to theses informations:

  • Your discord account information
  • The server where the bot it
  • The messages that are sent in each server I DO NOT STORE YOUR MESSAGES : they are only used if you enable the Auto Recognition feature, and not stored.

The only things that are stored are :

  • The server ID (to know what settings are enabled on the server)
